## 属性

| 参数           | 说明                | 类型   | 可选值 | 默认值              |
| -------------- | ------------------- | ------ | ------ | ------------------- |
| blockClassName | 块状元素的 class 名 | String | -      | editor-custom-block |
| 单元格         | 单元格              |

## 方法

| 参数             | 说明               | 参数                            |
| ---------------- | ------------------ | ------------------------------- |
| customBlockStyle | 自定义块状代码样式 | code【String】 块状代码字符串   |
| deleteBlockCode  | 删除块状代码触发   | codes【Array】 块状代码字符集合 |
